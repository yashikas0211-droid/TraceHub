About TraceHub

A simple Lost and Found platform for students and users.

ABOUT THE PROJECT

What is TraceHub?

TraceHub is a Lost and Found website where users can report lost items, report found items, and contact each other to recover their belongings. It is designed to be simple and easy to use for students and college users.

TECHNOLOGIES USED

HTML, CSS and JavaScript

This project is built using HTML for the structure, CSS for the styling and design, and JavaScript for the interactive features like form handling and page functionality.
